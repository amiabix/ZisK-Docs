<Summary>

Raw BLS12-381 elliptic curve syscalls. G1 point operands use
`SyscallPoint384`: `x ++ y`, each coordinate as 6 LE `u64` limbs
(384 bits). Fp2 operands use `SyscallComplex384`: `real ++ imaginary`,
6 LE `u64` limbs each. Results are written back into `p.x` in-place.

</Summary>

## G1 curve

<FnPanel>

<FnDef name="syscall_bls12_381_curve_add">

BLS12-381 G1 point addition. Reads two affine points from `p` and
writes the sum into `p.x` in-place.

```rust
pub unsafe fn syscall_bls12_381_curve_add(p: *mut SyscallPoint384)
```

#### Parameters

| Name | Type | Description |
|---|---|---|
| `p` | `*mut SyscallPoint384` | Input: `p.x` and `p.y` are the two addend points (6 LE `u64` limbs each). Output: result written back into `p.x`. |

#### Returns

Nothing. The result point is written into `p.x` in-place.

</FnDef>

<FnDef name="syscall_bls12_381_curve_dbl">

BLS12-381 G1 point doubling. Doubles the point in `p.x` and writes
the result back in-place.

```rust
pub unsafe fn syscall_bls12_381_curve_dbl(p: *mut SyscallPoint384)
```

#### Parameters

| Name | Type | Description |
|---|---|---|
| `p` | `*mut SyscallPoint384` | Input: `p.x` is the point to double (6 LE `u64` limbs). Output: result written back into `p.x`. |

#### Returns

Nothing. The doubled point is written into `p.x` in-place.

</FnDef>

</FnPanel>

## Fp2 arithmetic

back into `p.x`.

```rust
pub unsafe fn syscall_bls12_381_complex_add(p: *mut SyscallComplex384)
```

#### Parameters

| Name | Type | Description |
|---|---|---|
| `p` | `*mut SyscallComplex384` | Input: `p.x` (real) and `p.y` (imaginary) are the two Fp2 operands. Output: result written into `p.x`. |

#### Returns

Nothing. The result is written into `p.x` in-place.

</FnDef>
